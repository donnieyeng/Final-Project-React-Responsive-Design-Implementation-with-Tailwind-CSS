# ReTreadPro Landing Page

**ReTreadPro** is a responsive, modern landing page built with **React**, **Vite**, and **Tailwind CSS**. It showcases tyre retreading services, highlighting the company’s expertise, processes, and contact information in a user-friendly format.

## Project Overview

ReTreadPro specializes in restoring worn tyres using certified machinery and skilled technicians. This landing page provides:

- Clear introduction to the company and services  
- Detailed retreading process with visual highlights  
- About section outlining company mission and sustainability  
- Contact section with email, phone, location, and quote request  
- Fully responsive design with smooth scrolling  

---

## Features

- **Responsive Design** – Mobile-first and works seamlessly on all screen sizes.  
- **Sticky Header** – Fixed header with smooth scrolling to sections.  
- **Retreading Process** – Feature cards highlighting machines, technicians, and quality checks.  
- **About Section** – Engaging description of the company’s mission and services.  
- **Contact Section** – Easy-to-access contact info with call-to-action button.  
- **Footer** – Persistent footer with email, phone, and location links.  
- **Tailwind CSS** – Modern styling with utility-first approach.  

---

## Technologies Used

- **React** – Frontend UI library  
- **Vite** – Fast development server and build tool  
- **Tailwind CSS** – Utility-first CSS framework for styling  
- **JavaScript (ES6)** – Application logic

## Installation

```bash

cd retreadpro-landing
npm install
Ensure Node.js v20+ is installed (required by Vite).

Usage

npm run dev
